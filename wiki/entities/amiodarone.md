# Amiodarone

## What it is
Amiodarone is a medicine commonly used to treat some heart problems, and a recognised drug cause of hypothyroidism (see [Mina Shahnawaz – Tiredness](../sources/mina-shahnawaz-tiredness.md)).

## Properties
- **Type:** drug
- **Owner / Operator:** —
- **Status in our context:** listed as a medicine that can cause hypothyroidism.

## Description
- Commonly used to treat some heart problems (see [source](../sources/mina-shahnawaz-tiredness.md)).
- Named among medicines that can cause [hypothyroidism](./hypothyroidism.md) (see [source](../sources/mina-shahnawaz-tiredness.md)).

## Relationships
- Can cause: [hypothyroidism](./hypothyroidism.md)
